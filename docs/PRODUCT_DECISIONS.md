# 제품 결정

- 여러 계산기 대신 하나의 목표 도착일 내비게이션을 제공한다.
- 첫 결과는 오직 목표 금액, 지금까지 모은 돈, 매달 모을 돈과 0% 수익률로 계산한다.
- 월급·지출은 매달 모을 돈을 모르는 사용자가 직접 선택할 때만 묻는다.
- 수익률은 결과 이후의 선택적 가정이며 추천값으로 표현하지 않는다.
- 금액 차이보다 도착 기간과 연월, 선택으로 바뀌는 시간을 먼저 보여준다.
- 회원가입과 서버 저장 없이 브라우저 로컬 저장만 제공한다.
- 세금, 수수료, 물가, 개별 상품 특성은 MVP 계산에서 제외한다.
- 화면 용어는 `조건 바꿔보기`, `이번 달 업데이트`, `데이터 백업`을 사용한다.
