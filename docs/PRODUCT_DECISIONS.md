# 제품 결정

- INVETK는 목표 도착일 계산기가 아니라 목표 날짜의 부족분을 해결하는 도구다.
- 첫 단계에서 목표 금액과 날짜를 하나의 목적지 질문으로 묶고, 이후 지금까지 모은 돈과 매달 모을 돈을 묻는다.
- 첫 결과는 0% 수익률로 목표 날짜의 예상 금액과 부족분을 계산한다.
- 부족할 때는 월 적립 확대, 목돈 추가, 기간 조정의 세 가지 해결책을 만든다.
- 목표를 맞출 때는 현재 계획 유지, 월 적립 부담 낮추기, 현재 속도의 예상 도착일 확인을 제공한다.
- 선택한 실행안을 자동이체·이번 달 추가금·월말 확인으로 구성된 행동 계획으로 바꾼다.
- 만 원 입력은 즉시 원 단위로 다시 보여 단위 착오를 줄이고, 이전 단계의 핵심 입력은 다음 단계 상단에 유지한다.
- 해결책은 `목돈 없이`, `월 부담 유지`, `기간 조정`처럼 차이를 먼저 보여주며 선택 결과를 행동 계획에 즉시 반영한다.
- 이번 달 행동 계획은 3단계 체크리스트와 완료 진행률로 제공한다.
- 월급·지출은 매달 모을 돈을 모르는 사용자가 직접 선택할 때만 묻는다.
- 수익률은 결과 이후의 선택적 가정이며 추천값으로 표현하지 않는다.
- 결과의 우선순위는 예상 목표 충족률과 부족분, 해결책 3개, 이번 달 행동 계획, 입력값, 조건 비교와 상세 계산 순서다.
- 회원가입과 서버 저장 없이 브라우저 로컬 저장만 제공한다.
- 세금, 수수료, 물가, 개별 상품 특성은 MVP 계산에서 제외한다.
- 화면 용어는 `조건 바꿔보기`, `이번 달 업데이트`, `데이터 백업`을 사용한다.
