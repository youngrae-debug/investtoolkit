# 검증 계획

## 핵심 가설

- 첫 방문자가 60초 안에 세 단계 입력을 완료할 수 있다.
- 결과에서 목표 날짜의 부족분을 즉시 이해할 수 있다.
- 실행안 3개 중 하나를 선택하고 이번 달 행동 계획을 이해할 수 있다.
- 만 원 입력 뒤 원 단위 환산값을 확인해 단위 오류를 스스로 발견할 수 있다.
- 실행안을 바꾸면 행동 계획이 즉시 바뀌고 체크리스트 완료 상태를 확인할 수 있다.
- 월 적립액 변화가 목표일에 미치는 시간 차이를 이해할 수 있다.
- 저장이 서버가 아닌 현재 브라우저에서만 이뤄짐을 이해한다.
- 저장된 계획을 업데이트했을 때 화면의 실행안과 다음 계산의 월 적립액이 일치한다.
- 저장 데이터가 없는 브라우저에서도 백업 파일로 계획을 복원할 수 있다.
- 월급·지출 도우미를 키보드만으로 열고 이동하고 닫을 수 있다.

## 비민감 이벤트

`gps_started`, `gps_step_completed`, `gps_calculation_completed`, `goal_action_selected`, `feasibility_limits_applied`, `monthly_action_plan_copied`, `condition_compared`, `plan_saved`, `monthly_update_saved`, `result_copied`, `web_share_used`, `share_card_created`, `backup_exported`, `backup_imported`, `local_data_deleted`를 사용한다. 금액, 소득, 자산, 지출, 메모, 계산 결과는 속성에 넣지 않는다.

## 자동 검증 범위

- 단위 테스트: 월별 계산, 목표일 해결안, 조건 이벤트, 저장 마이그레이션, 한국어 포맷
- 계산 일치성: 연 -20%, 0%, 5%, 30%에서 목표일 공식과 월별 시뮬레이터 비교
- 모바일 E2E: 3단계 입력, 입력 범위, 실행안 선택, 저장·복원, 가능한 범위, 이번 달 업데이트, 백업, 음수 현금흐름, 목표 달성, 수익률 변경, 모달 포커스, 가이드 광고 동의와 핵심 도구 광고 미노출
- 완료 검사: `npm run typecheck`, `npm run lint`, `npm test`, `npm run test:e2e`, `npm run build`

## 실제 사용자 검증

### 참가자와 원칙

- 돈 목표를 직접 세워본 성인 5명을 목표로 하되, 첫 세션은 1명부터 시작한다.
- 실제 소득·자산·부채를 묻거나 기록하지 않고 아래 테스트용 금액만 사용한다.
- 진행자는 화면의 정답이나 다음 버튼 위치를 먼저 설명하지 않는다.
- 참가자에게 생각나는 내용을 소리 내어 말해 달라고 요청하고, 막힌 지점과 표현을 그대로 기록한다.

### 공통 테스트 시나리오

- 목표: 5년 뒤 1억 원
- 현재 모은 돈: 3,000만 원
- 매달 모을 돈: 100만 원
- 정책 혜택 확인용 조건: 만 29세, 근로소득, 지난해 총급여 3,600만 원, 1인 가구, 가구소득 3,600만 원, 복지 자격 해당 없음

### 진행 과제

1. 첫 화면만 보고 이 서비스가 무엇을 해주는지 설명한다.
2. 공통 시나리오로 계산을 완료한다.
3. 부족분과 세 가지 해결책의 차이를 설명한다.
4. 본인이라면 선택할 해결책 하나를 고르고 이번 달 할 일을 말한다.
5. 계획을 저장한 뒤 첫 화면으로 돌아가 저장된 계획을 다시 연다.
6. 현재 모은 돈을 3,100만 원으로 바꾸어 이번 달 기록을 저장한다.
7. 데이터 백업 기능을 찾아 백업 파일을 만든다.
8. 정책 혜택 페이지에서 공통 조건으로 대상 가능성을 확인한다.

### 성공 기준

- 도움 없이 60초 안에 3단계 계산을 완료한다.
- 목표일의 예상 금액과 부족분을 반대로 이해하지 않는다.
- 월 적립, 시작 자금, 기간 조정의 차이를 자기 말로 설명한다.
- 실행안 선택 뒤 이번 달 행동 계획이 바뀐 것을 알아차린다.
- 저장 위치가 계정이나 서버가 아닌 현재 브라우저임을 이해한다.
- 정책 결과를 가입 확정이 아닌 대상 가능성 확인으로 이해한다.
- 각 과제의 성공 여부, 소요 시간, 막힌 지점, 참가자 표현을 기록한다.
