# 구현 계획

최종 업데이트: 2026-07-15

## 제품 우선순위

첫 방문자는 세 단계로 목표 금액·날짜, 지금까지 모은 돈, 매달 모을 돈을 입력한다. 60초 안에 원금 기준 부족분과 실행안 3개, 이번 달 행동 계획을 확인한다. 월 적립액을 모를 때만 월급과 지출 보조 흐름을 사용한다. 수익률, 조건 비교, 상세 그래프, 저장은 첫 결과 이후에만 제공한다.

## 구현 순서

1. 순수 함수 기반 월별 시뮬레이션 엔진과 한국어 포맷 함수
2. 1/3, 2/3, 3/3 단계형 목표·현재·월 적립 입력
3. 목표 날짜 기준 부족분과 월 적립·시작 자금·혼합 실행안 계산
4. 선택한 실행안으로 이번 달 행동 계획 생성
5. 선택적 수익률 가정과 조건 바꿔보기
6. 명시적 localStorage 저장, 이번 달 업데이트, 데이터 백업
7. 공유, 상세 그래프·계산 내역, 계산 기준과 법적 문서
8. 가이드, 메타데이터, sitemap, robots, 접근성
9. 단위·E2E 테스트와 프로덕션 빌드

## 데이터 경계

모든 금융 계산은 브라우저에서 수행한다. 분석 이벤트에는 단계 번호와 상태 같은 비금융 속성만 허용한다. 저장된 계획만 목표 날짜를 포함한 버전 2 스키마로 localStorage에 보관하며 버전 1은 자동 이전한다.
