# 구현 계획

최종 업데이트: 2026-07-15

## 제품 우선순위

첫 방문자는 세 가지 질문만으로 60초 안에 원금 기준 목표 도착일을 확인한다. 월 적립액을 모를 때만 월급과 지출 보조 흐름을 사용한다. 수익률, 조건 비교, 상세 그래프, 저장은 첫 결과 이후에만 제공한다.

## 구현 순서

1. 순수 함수 기반 월별 시뮬레이션 엔진과 한국어 포맷 함수
2. 1/3, 2/3, 3/3 단계형 금액 입력과 0% 첫 결과
3. 선택적 수익률 가정과 조건 바꿔보기
4. 명시적 localStorage 저장, 이번 달 업데이트, 데이터 백업
5. 공유, 상세 그래프·계산 내역, 계산 기준과 법적 문서
6. 가이드, 메타데이터, sitemap, robots, 접근성
7. 단위·E2E 테스트와 프로덕션 빌드

## 데이터 경계

모든 금융 계산은 브라우저에서 수행한다. 분석 이벤트에는 단계 번호와 상태 같은 비금융 속성만 허용한다. 저장된 계획만 버전 1 스키마로 localStorage에 보관한다.
